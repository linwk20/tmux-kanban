# Claude Design Style

Design tokens and guidelines used by tmux-kanban's UI, based on the Claude (Anthropic) design system.

Based on: [VoltAgent/awesome-design-md - Claude Design System](https://github.com/VoltAgent/awesome-design-md/tree/main/design-md/claude)

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | Complete design system documentation (colors, typography, spacing, elevation, components) |
| `preview.html` | Interactive design token catalog (light mode) |
| `preview-dark.html` | Interactive design token catalog (dark mode) |

## Usage

Feed `DESIGN.md` to any AI coding agent (Claude, Cursor, Codex) when building or modifying the tmux-kanban UI. The design tokens ensure consistent visual language across the application.

Open `preview.html` or `preview-dark.html` in a browser to see all design tokens visualized.
